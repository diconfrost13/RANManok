========================================================================
    정적 라이브러리 : minLib 프로젝트 개요
========================================================================

- 가능 하면 플랫폼에 독립적으로 작성한다.
- 3D 코드나 랜더링 코드등은 작성하지 않는다.
- 범용적으로 사용가능한 라이브러리만 제작한다.
- namespace minLib 안에 모든 라이브러리를 작성한다.

========================================================================
위치설명
========================================================================
Math / 수학과 관계된 범용 클래스를 작성한다.
Container / 자료구조에 관한 범용 클래스를 작성한다.
Compress / 압축과 관련된 범용 클래스를 작성한다.

========================================================================
초기화
========================================================================
라이브러리의 초기화는 minLib::initLib() 을 호출한다.
